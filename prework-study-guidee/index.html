HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element is essential for search engine optimization, or SEO.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • An asterisk * is used as a "wild card" selector to include all elements on the page.
  • Two selectors separated by a comma can share on line of code if they require the same styling.
  • Color can be introduced on a style sheet a number of ways, including hex # or semantic reference.
  • A . denotes a class selector.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone [url]: retrieves an entire repository from a hosted location via URL.
  • git add [file]: add a file as it looks now to your next commit.
  • git add -A: -A is used to commit ALL files in the working branch.
  • git commit -m "desc. message": -m associates a message with our commit. These should be concise, but descriptive.
  • git pull origin main: pulls the base/main branch in order to sync local with remote branches. If branches are in sync, "Already up to date" will appear.
  • git push origin "branch name": pushes up all branch changes
